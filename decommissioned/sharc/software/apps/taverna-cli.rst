Taverna is an open source and domain-independent Workflow Management System â€“ a suite of tools used to design and execute scientific workflows and aid in silico experimentation. Taverna-cli is the command line version (no GUI).

Usage
-----

The lastest version of Taverna-cli can be activated using the module file::

    module load apps/taverna/cli

Alternatively, you can load a specific version of Taverna-cli::

    module load apps/taverna/cli/2.5.0
